# Portfolio Website

Clean, responsive portfolio built with HTML, CSS, and vanilla JavaScript.

## Features

- Fully responsive design
- Auto-scrolling skills carousel
- Project showcase with GitHub links
- Interactive social links

## Setup

Clone and open `index.html` in your browser.

## Customization

- Update info in `index.html`
- Change colors in `:root` variables (`main.css`)
- Replace images in `assets/images/`
